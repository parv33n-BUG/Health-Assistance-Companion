# Health-Assistance-Companion

MedVerse AI: Your personal Hinglish-speaking health companion. This tool helps you analyze medical reports (PDF/Images) and provides easy-to-understand health insights through an interactive dashboard.

## 🚀 Key Features
* **Conversational AI**: Get comforting, professional health advice in Hinglish.
* **Smart Analysis**: Upload medical report cards (PDF or Images) for instant evaluation.
* **Visual Dashboards**: Track your health metrics with a clean, built-in dashboard.
* **Privacy Focused**: Automatic sensitive name redaction in AI responses.

## 🛠️ Tech Stack
* **Language**: Python
* **AI Engine**: Google Gemini Pro
* **Frontend/UI**: Gradio
* **Data Processing**: PyPDF, PIL (Pillow)

## 📋 Prerequisites
Make sure you have the following installed:
- Python 3.x
- Google Colab (recommended for easy setup)
- A Google Gemini API Key
